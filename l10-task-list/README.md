# Project #1 - Laravel 10 Task List

#### Commands

- Run MySQL
    - docker compose up
    - `http://localhost:8080/?server=mysql`

- Prepare Database and Run migrations
    - php artisan migrate

- Make Model and Migration file
    - php artisan make:model `ModelName` -m

- Run Migration
    - php artisan migrate

- Rollback Migration
    - php artisan migrate:rollback
